الأسئلة الشائعة
================


كيف أجد عنوان IP الخاص بـ PLC؟
---------------------------------

عنوان IP الخاص بـ PLC يتم تكوينه في مشروع PLC (مثل TIA Portal أو STEP 7).
يمكنك أيضاً إيجاده عبر:

1. التحقق من شاشة PLC (إذا كانت موجودة)
2. مسح شبكتك بأدوات مثل ``nmap``
3. التحقق من إعدادات المشروع في برنامج الهندسة


ما هي أرقام الراك والفتحة (Rack/Slot)؟
-----------------------------------------

- **الراك (Rack)**: الحامل الفعلي الذي يتم تركيب PLC عليه (عادة ``0``)
- **الفتحة (Slot)**: موقع وحدة المعالجة في الراك (عادة ``0`` لـ S7-1200/1500،
  ``2`` لـ S7-300/400)

الإعدادات الشائعة:

.. list-table::
   :header-rows: 1

   * - سلسلة PLC
     - الراك
     - الفتحة
   * - S7-1200
     - 0
     - 0
   * - S7-1500
     - 0
     - 0
   * - S7-300
     - 0
     - 2
   * - S7-400
     - 0
     - 2 أو 3


هل يمكنني الاتصال بعدة أجهزة PLC في نفس الوقت؟
--------------------------------------------------

نعم! أنشئ عدة نسخ من ``Client``:

.. code-block:: python

    import snap7

    client1 = snap7.Client()
    client1.connect("192.168.0.1", 0, 0)

    client2 = snap7.Client()
    client2.connect("192.168.0.2", 0, 0)


هل يدعم python-snap7 أجهزة S7-1200 و S7-1500؟
--------------------------------------------------

نعم، لكن تحتاج لتفعيل اتصال **PUT/GET** على PLC:

1. افتح TIA Portal
2. اذهب إلى خصائص وحدة المعالجة
3. انتقل إلى **الحماية والأمان** ← **آليات الاتصال**
4. فعّل **السماح بالوصول عبر اتصال PUT/GET**


ما أنواع البيانات المدعومة؟
-----------------------------

يدعم python-snap7 قراءة وكتابة:

- **Bool** — بتات مفردة
- **Byte/Int/DInt/Real** — أنواع رقمية
- **Word/DWord** — أعداد صحيحة بدون إشارة
- **String** — سلاسل نصية
- **Date/Time/DateTime** — أنواع زمنية
- **S5Time/Time** — تنسيقات الوقت الخاصة بـ PLC


هل python-snap7 آمن لتعدد الخيوط (Thread-safe)؟
---------------------------------------------------

يجب استخدام كل نسخة ``Client`` من خيط واحد فقط. للتطبيقات متعددة الخيوط،
أنشئ ``Client`` واحد لكل خيط أو استخدم مزامنة الخيوط.


ما الحجم الأقصى للبيانات التي يمكنني قراءتها/كتابتها؟
---------------------------------------------------------

يعتمد حجم البيانات الأقصى على حجم PDU المتفاوض عليه مع PLC:

- **S7-300**: حتى 240 بايت لكل PDU
- **S7-400/1200**: حتى 480 بايت لكل PDU
- **S7-1500**: حتى 960 بايت لكل PDU

للبيانات الأكبر، استخدم ``read_multi_vars()`` أو قسّم القراءة لعدة أجزاء.
